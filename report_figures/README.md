# Report Figures

This directory contains figure source files for the Part 5 report.

- `figure01_housing_scientist_flow.mmd`
  Overall system flow from user request to inquiry drafting and contract risk check.
- `figure02_ai_scientist_v2_mapping.mmd`
  Concrete mapping between AI-Scientist-v2 and The AI Housing Scientist.
- `figure03_branch_selection_example.mmd`
  Example of branch-based search and why final selection is not a simple highest-count choice.
